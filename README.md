## Como executar

- Baixe e instale o Vagrant: http://www.vagrantup.com/downloads.html

- Na linha de commando, clone este repositório:
````
$ git clone https://github.com/chatcaos/docker-hands-on
````
- Execute:
````
$ vagrant up
````
- Acesse o ambiente com:
````
$ vagrant ssh
````
